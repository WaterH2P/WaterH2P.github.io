使用 Hexo 搭建 Blog

此文介绍使用 Hexo + Github 架构搭建个人博客的详细步骤
最近更新：2018-09-28
操作系统：MacOS 10.13.6

【转自 Shelton Lee 的博客】Shelton Lee

本地博客仓库

  • 安装 Node

  • 安装 Hexo

  • 初始化博客

    在本地建立一个文件夹作为存放博客的仓库（名字任意），例如 MyBlog 。

    进入文件夹，执行初始化。

  • 生成静态文件

    使用 hexo g 或 hexo generate 命令生成静态页面。

    • 本地启动服务器

      使用 hexo s 或 hexo server 启动本地服务器，可以在本地查看、调试博客。

    • 浏览器输入 http://localhost:4000 即可查看

    • 使用组合键 ctrl + c 关闭本地服务器

  • 监视文件变动
