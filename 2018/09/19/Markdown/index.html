一、数学公式


1. 插入公式

或


2. 输入上下标

输入上标使用：^，输入下标使用：_。

例如：

显示 ：$ x_0^+ $


3. 括号

(),[],| 可以直接使用标识符号本身，\{\} 来表示大括号。当要显示大号的括号或分隔符时，要用 \left 和 \right 命令。

例如：

显示：$ f(x) = \left( \dfrac{x - \ln x}{x} \right)^2 $

如果一边符号不需要显示时使用 \left. 或 \right. 。

例如：

显示：$ \left. \dfrac{ {\rm d}^2 y}{ {\rm d}x^2 } \right|_{t = 0} $


4. 输入分数

通常使用 \frac {分子} {分母} 命令产生一个分数，分数可嵌套。

便捷情况可直接输入 \frac ab 来快速生成一个 $ \frac ab $ 。

也可以使用 \dfreac 代替 \frac ，使得分式看起来更好看一点。

例如：

显示：$ \frac{a-1}{b-1} \quad \text{and} \quad \dfrac{a-1}{b-1} $


5. 输入开方

使用 \sqrt [根指数] {被开方数} 命令输入开方（如果不输入根指数，默认为2）。

例如：

显示：$ \sqrt{1 + x^2} \quad \text{and} \quad \sqrt[n]{ax + b} $


6. 输入省略号

\cdot 可以表示乘号， \ldots 表示与文本底线对齐的省略号，\cdots 表示与文本中线对齐的省略号。

例如：

显示：$ f(x_1,x_2, \ldots,x_n) = k \cdot x_1^2 + x_2^2 + \cdots + x_n^2 $


7. 输入积分

使用 int_积分下限^积分上限 {被积表达式} 来输入一个积分。

例如：

显示：$ \int_a^b f(x) {\rm d}x $


8. 输入极限

使用 \lim_{变量 \to 表达式} 表达式 来输入一个极限，也可以使用 \lim\limits_{变量 \to 表达式} 表达式 将条件放到极限符号下面。

例如：

显示：$ \lim_{x \to x_0}f(x) = A \quad \text{and} \quad \lim\limits_{x \to x_0}f(x) = A $
