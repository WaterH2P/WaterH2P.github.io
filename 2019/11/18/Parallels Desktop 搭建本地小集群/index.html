Parallels 安装虚拟机

  • 我安装的是 CentOS 7


安装 Parallels Tools

  • 虚拟机和本机实现文件共享需要安装 Parallels Tools

  1. 设置虚拟机 Hardware 下 CD/DVD 中的 Source 为 prl-tools-lin.iso。

  2. 进入到虚拟机 cd /media 目录下，创建文件夹 mkdir cdrom。

  3. 执行命令 mount -o exec /dev/cdrom /media/cdrom。

    • 看到 mount: /dev/sr0 is write-protected, mounting read-only 表示挂载成功。

  4. 进入到虚拟机 cd /media/cdrom 目录下，执行 ./install，然后一直 Next，直到开始安装。

  5. 安装成功后执行 reboot 重启虚拟机。

  6. 设置虚拟机 Options 下 Sharing 中的 Custom Folders，添加本机共享的文件夹，例如下图的 Linux。

  7. 进入到虚拟机 cd /media/psf 下查看是否有共享的文件夹。

  8. 还可以设置软链接方便文件夹的访问。
